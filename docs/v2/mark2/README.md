# Mark 2

## Nome
Advanced Robotics Platform

## Função do Mark 2
O Mark 2 não é a implementação desta rodada.  
Ele é a **visão arquitetural completa do próximo nível**, usada para guiar decisões do Mark 1.

## Objetivo
Expandir o sistema de "drone control platform" para "plataforma robótica modular" com:
- múltiplas famílias de veículo
- payloads plugáveis
- actuators plugáveis
- capabilities formais
- readiness para IA/MCP
- readiness para hardware real
