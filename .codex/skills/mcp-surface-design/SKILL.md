# MCP Surface Design

## When to use
Quando a tarefa envolver interface futura para IA, capabilities ou machine API.

## Goal
Preparar uma surface segura e formal para futura integração MCP.

## Inputs required
mark2 docs, action model, capability model

## Output expectations
tool families, action mappings, docs de readiness

## Rules
MCP futuro fala com control plane

## Anti-patterns
expor flight core cru para IA

## Validation
revisão arquitetural, coerência de action/capability surface
