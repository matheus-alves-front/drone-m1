# drone_base

Placeholder do modelo base do veiculo para a Fase 1.

## Intencao

- Dar uma identidade inicial ao veiculo na cena.
- Servir como contrato de organizacao antes do modelo real do drone.
- Manter o asset simples enquanto o fluxo de simulacao ainda esta sendo fechado.

## Limite

- Nenhum comportamento de voo.
- Nenhum tuning fisico final.
- Nenhuma dependencia de hardware real.
