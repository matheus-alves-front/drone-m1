# Models

Diretorio reservado para modelos reutilizaveis da simulacao.

## Intencao

- Conter o veiculo base e objetos auxiliares.
- Permitir reutilizacao entre worlds e cenarios.

## Modelo inicial

- `drone_base/` representa o placeholder do veiculo na Fase 1.
- O modelo existe para dar estrutura ao monorepo antes de assets mais ricos.
- A evolucao para um modelo mais completo deve acontecer sem misturar comportamento de missao.

## Regras

- Um modelo por diretorio quando o asset crescer.
- Nomear claramente a origem e a finalidade do modelo.
- Nao colocar aqui logica de controle ou comportamento de missao.
