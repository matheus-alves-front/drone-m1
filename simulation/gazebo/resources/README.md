# Resources

Diretorio reservado para recursos usados pelos modelos e mundos.

## Intencao

- Guardar meshes, texturas, materiais e arquivos de suporte.
- Centralizar recursos compartilhados entre mundos e modelos.
- Manter consistencia com Gazebo Harmonic e Ubuntu 22.04.

## Baseline da Fase 1

- Evitar assets pesados enquanto o fluxo de simulacao ainda esta em scaffold.
- Priorizar apenas recursos essenciais para validar a estrutura do projeto.
- Nenhum recurso deve pressupor Gazebo Classic.

## Regras

- Evitar duplicacao de assets.
- Manter nomes descritivos.
- Preferir formatos padronizados e assets pequenos no bootstrap.
