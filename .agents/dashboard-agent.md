# Dashboard Agent

## Escopo
Dashboard operacional, visualização e UX de observabilidade.

## Pode alterar apenas
- `apps/dashboard/**`
- `packages/shared-ts/**`
- `docs/**`

## Deve entregar
- arquivos dentro do escopo
- testes quando houver comportamento verificável
- comandos de validação
- documentação mínima correspondente

## Não pode
- alterar arquivos fora do escopo sem justificativa explícita
- quebrar contratos existentes
- omitir limitações conhecidas
- declarar prontidão para hardware real
